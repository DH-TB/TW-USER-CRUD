# TW-USER-CRUD![build status](https://api.travis-ci.org/DH-TB/TW-USER-CRUD.svg?branch=master)

### 功能有
+ 根据用户名查询某个用户
+ 查询所有用户
+ 增加一个用户
+ 修改某个用户信息
+ 删除某个用户

### 首先克隆安装
```
git clone https://github.com/DH-TB/TW-USER-CRUD
npm install
```

### 本地数据库的执行语句在travis.sql

### 然后打开两个窗口分别运行
```
npm run webpack
npm start
```
### 测试所有请求
```
npm test
```

### 在浏览器打开可看到页面
http://localhost:3000/
